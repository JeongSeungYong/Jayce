절대 경로 사용하기

절대 경로란 도메인을 포함한 디랙토리 루트 폴더부터 원하고자 하는 목적지 까지의 주소를 절대 경로라고 한다.
도메인이 www.homepage.com이라고 가정해보자 그럴때 내가 연결하고자 하는 디랙토리가 main_file/image 라고 할때, 절대 결로는 www.homepage.com/main_file/image 가 된다.
이 절대 경로는 요소와 함께 사용해 주로 다른 홈페이지를 링크를 걸때 사용하는데, 지금 보시는 제 페이지를 자신의 홈페이지에 링크를 건다면,
<a href="https://jeongseungyong.github.io/Jayce">제이스의 일지</a>
이렇게 적으면 제 홈페이지가 제 절대경로의 주소에 의해 여러분의 홈페이지에 링크가 걸려질것입니다.




id 속성을 사용해 목적지 생성하기

예를 들어 우리가 한페이지에 300줄 이상의 긴 문장이 담긴 홈페이지를 만들었다고 합시다.
그곳 중간 중간에 우리가 원하는 그곳만을 찾아보고 싶은 곳이 있을겁니다.
그런곳에 id속성에 값을 부여해 참조할때 쓰는 색인 기능처럼 사용합니다.
우리가 만든 홈페이지 어떤 곳에 식별자를 부여해 그곳으로 링크를 걸어 이동할수 있습니다.

예) <h3>이곳은 이동하고 싶은 소제목 입니다</h3>
여기에 여러분이 이동하고 싶은 소제목이 있습니다.
id 속성은 < > 태그 안에 어디든지 속성으로 들어가 목적지를 만들수 있습니다.
<h3 id="here"> 이곳은 이동하고 싶은 소제목 입니다</h3>
위의 코드에 h3요소 안에 here이라는 식별자를 생성했습니다.

id속성을 가진 요소와 연결 하는 방법

위에서는 id속성을 사용해 목적지를 생성했는데요.
이번에는 그 생성한 목적지로 연결 하는 방법을 알아 보겠습니다.
그 방법은 간단 합니다. <a>요소의 경로 뒷편에 #과 이동하고자 하는 식별자를 적어 넣으면 됩니다.
예) <a href="jeongseungyong.github.io/Jayce/index.html#here>여러분이 이동하고 싶은 소제목으로 이동합니다</a>
위의 링크를 클릭하면 입력된 식별자를 가진 목적지로 링크를 통해 이동하게 됩니다.
